# Machine-Learning-Model-Implementation-CSE427-Project-
This project analyzed the relationship between field of study and occupation using machine learning techniques. A complete pipeline was developed, including data preprocessing, model training, and evaluation, using Logistic Regression, Random Forest, and AdaBoost on a dataset of 38,444 records with 23 variables.
